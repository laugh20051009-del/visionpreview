# DSD Experience Space 사용 설명서

## 1. 공간 개요

DSD Experience Space는 브라우저에서 실행되는 3D 체험 공간입니다. 사용자는 가상 공간 안을 둘러보며 원하는 위치에 웹 화면이나 동영상 화면을 띄우고, 크기와 곡률을 조절해 Vision Pro 스타일의 공중 디스플레이 경험을 확인할 수 있습니다.

이 공간은 다음 기능을 제공합니다.

- 3D 공간 탐색
- Google, Coupang, Wikipedia, Naver Dict, JSFiddle 바로가기 화면 생성
- 직접 입력한 URL 화면 표시
- 동영상 URL을 이용한 대형 곡면 스크린 생성
- 생성한 화면의 위치 이동, 크기 조절, 삭제

## 2. 실행 방법

1. `index_air_icons.html` 파일을 크롬, 엣지 등 최신 브라우저에서 엽니다.
2. 화면 중앙의 안내 오버레이가 보이면 아무 곳이나 클릭해 닫습니다.
3. 하단의 `screen`, `video mode` 버튼을 사용해 체험을 시작합니다.

인터넷 연결이 필요합니다. 3D 엔진인 A-Frame이 외부 CDN에서 로드되고, 생성되는 웹 화면도 실제 웹사이트를 불러오기 때문입니다.

## 3. 기본 조작

### 시점 이동

- 마우스 드래그: 주변 둘러보기
- `W`, `A`, `S`, `D`: 앞, 왼쪽, 뒤, 오른쪽 이동
- 브라우저 창 크기 변경: 화면 비율에 맞춰 자동 조정

카메라 높이는 고정되어 있어 일반적인 서 있는 시점으로 공간을 탐색합니다.

### 안내 화면 닫기

처음 진입하면 환영 안내 화면이 표시됩니다. 화면을 한 번 클릭하면 안내 화면이 사라지고 체험 공간 조작이 가능합니다.

## 4. 웹 화면 만들기

1. 하단의 `screen` 버튼을 클릭합니다.
2. 화면 위에 바로가기 아이콘 팔레트가 나타납니다.
3. 원하는 아이콘을 클릭합니다.
   - Google
   - Coupang
   - Wikipedia
   - Naver Dict
   - JSFiddle
4. 선택한 웹사이트가 사용자의 정면에 공중 화면으로 생성됩니다.

생성된 화면을 클릭하면 해당 화면이 선택되고, 오른쪽 설정 패널에서 크기와 URL을 조정할 수 있습니다.

## 5. 직접 URL 입력하기

1. 생성된 화면을 클릭해 선택합니다.
2. 오른쪽 설정 패널의 URL 입력칸에 주소를 입력합니다.
3. `Go` 버튼을 누르거나 Enter 키를 누릅니다.

주소 입력 시 `https://`를 생략해도 자동으로 보정됩니다. 예를 들어 `wikipedia.org`를 입력하면 `https://www.wikipedia.org` 형태로 열립니다.

## 6. 화면 크기 조절

화면을 선택하면 오른쪽 설정 패널이 나타납니다.

- `width`: 화면 너비 조절
- `height`: 화면 높이 조절
- `curve`: 화면 휘어짐 조절

일반 웹 화면에서는 `curve`를 낮게 조절해 평면 또는 약한 곡면 화면으로 사용할 수 있습니다. 동영상 모드에서는 더 큰 곡률 범위가 적용되어 극장형 곡면 스크린처럼 보입니다.

## 7. 화면 이동하기

1. 이동할 화면을 클릭해 선택합니다.
2. 화면 위에 표시되는 이동 핸들을 누른 상태로 마우스를 움직입니다.
3. 원하는 위치에서 마우스를 놓으면 해당 위치에 화면이 고정됩니다.

화면은 사용자의 시선 방향을 기준으로 배치되며, 항상 보기 쉬운 방향으로 정렬됩니다.

## 8. 화면 삭제하기

1. 삭제할 화면을 클릭해 선택합니다.
2. 화면 오른쪽 위에 나타나는 `x` 버튼을 클릭합니다.
3. 선택한 화면이 공간에서 제거됩니다.

## 9. 동영상 모드 사용하기

1. 하단의 `video mode` 버튼을 클릭합니다.
2. 기본 예시 동영상이 곡면 스크린으로 생성됩니다.
3. 오른쪽 설정 패널에서 동영상 URL을 바꿀 수 있습니다.

동영상 모드는 `.mp4`, `.webm`, `.ogg` 형식의 직접 파일 URL을 지원합니다. 일반 웹페이지 주소를 동영상 모드에 입력하면 적용되지 않습니다.

## 10. 운영 및 시연 팁

- 체험 시작 전 브라우저를 전체 화면으로 전환하면 몰입감이 좋아집니다.
- 여러 개의 화면을 만들어 비교 시연할 수 있습니다.
- 주요 화면은 사용자의 정면에 배치하고, 보조 화면은 좌우에 배치하면 공간 활용이 자연스럽습니다.
- 동영상 스크린은 일반 웹 화면보다 크게 설정하면 시연 효과가 좋습니다.
- 외부 웹사이트는 사이트 정책에 따라 iframe 표시가 제한될 수 있습니다. 화면이 비어 보이면 다른 사이트나 직접 접근 가능한 콘텐츠 URL을 사용합니다.

## 11. 문제 해결

### 화면이 검게 보이거나 3D 공간이 열리지 않는 경우

- 인터넷 연결을 확인합니다.
- 최신 크롬 또는 엣지 브라우저에서 다시 엽니다.
- 회사 또는 학교 네트워크에서 외부 CDN 접속이 차단되어 있지 않은지 확인합니다.

### 웹사이트가 공중 화면 안에 표시되지 않는 경우

- 일부 사이트는 보안 정책 때문에 iframe 표시를 차단합니다.
- 다른 URL을 입력하거나, iframe 표시를 허용하는 콘텐츠를 사용합니다.

### 동영상이 재생되지 않는 경우

- URL이 `.mp4`, `.webm`, `.ogg`로 끝나는 직접 동영상 파일인지 확인합니다.
- 해당 동영상 서버가 브라우저 재생과 외부 접근을 허용하는지 확인합니다.

### 로고 이미지가 보이지 않는 경우

- `로고` 폴더가 `index_air_icons.html` 파일과 같은 위치에 있는지 확인합니다.
- `google.png`, `coupang.png` 파일명이 변경되지 않았는지 확인합니다.

## 12. 파일 구성

- `index_air_icons.html`: 3D 체험 공간 실행 파일
- `로고/google.png`: Google 바로가기 아이콘 이미지
- `로고/coupang.png`: Coupang 바로가기 아이콘 이미지
- `사용_설명서.md`: 본 사용 설명서
