# VisionSpace Preview 프로젝트 설명 및 포스터 구성안

## 1. 프로젝트 개요

**VisionSpace Preview**는 Apple Vision Pro의 공간 컴퓨팅 경험을 실제 기기 착용 전 웹 브라우저에서 미리 체험할 수 있도록 제작한 3D 인터랙티브 프로젝트이다. 사용자는 가상 공간 안을 둘러보며 웹 화면과 동영상 화면을 공중에 배치하고, 화면의 크기와 위치를 조절하면서 Vision Pro가 제공하는 몰입형 사용 방식을 간접적으로 경험할 수 있다.

이 프로젝트는 단순한 제품 소개 페이지가 아니라, 제품의 핵심 경험인 **공간 속 콘텐츠 배치**, **몰입형 미디어 감상**, **다중 화면 사용**, **직관적 인터페이스 조작**을 웹 환경에서 시연하는 체험형 프로토타입이다.

## 2. 제작 목적

- Apple Vision Pro의 공간 컴퓨팅 개념을 사용자가 쉽게 이해하도록 돕는다.
- 실제 제품을 착용하지 않아도 가상 공간 안에서 주요 사용 경험을 미리 확인할 수 있게 한다.
- 제품 소개 페이지와 3D 체험 공간을 연결하여 정보 탐색에서 직접 체험으로 이어지는 흐름을 만든다.
- 구매 전 사용자가 제품의 가능성과 몰입감을 상상할 수 있도록 시각적, 조작적 경험을 제공한다.

## 3. 전체 구성

프로젝트는 크게 두 영역으로 구성된다.

### 3.1 소개형 랜딩 페이지

루트의 `index.html`은 Apple Vision Pro 소개 페이지 형식으로 제작되었다. 상단에는 Apple 스타일의 글로벌 내비게이션과 제품 내비게이션이 있으며, 메인 화면에는 영상 또는 제품 이미지를 활용한 히어로 섹션이 배치되어 있다.

주요 구성은 다음과 같다.

- Apple Vision Pro 브랜드 이미지와 핵심 문구
- 공간 컴퓨팅을 설명하는 소개 문장
- VisionSpace Preview 개요 이미지
- 프로젝트 설명 섹션
- 체험공간 사용 방법을 보여주는 카드형 이미지 섹션
- 체험하기 및 구매하기 버튼

### 3.2 3D 체험 공간

`DSD_체험공간/index_air_icons.html`은 A-Frame 기반의 3D 체험 공간이다. 사용자는 검은 배경의 가상 공간에서 시점을 이동하며 공중 화면을 생성하고 조작할 수 있다.

주요 구성은 다음과 같다.

- A-Frame 기반 3D 씬
- 마우스 시점 이동 및 WASD 이동
- 안내 오버레이
- 공중 웹 화면 생성 기능
- 동영상 모드 화면 생성 기능
- 화면 크기, 높이, 곡률 조절 패널
- Google, Coupang, Wikipedia, Naver Dict, JSFiddle 바로가기 아이콘
- Apple 로고 바닥 마크와 은은한 원형 플로어 연출
- Apple 구매 페이지 연결 버튼

## 4. 핵심 기능

### 4.1 3D 공간 탐색

사용자는 브라우저에서 3D 공간을 둘러볼 수 있다. 마우스로 시점을 회전하고, WASD 키로 전후좌우 이동이 가능하다. 카메라 높이는 고정되어 있어 실제 사용자가 서서 공간을 바라보는 듯한 안정적인 시점을 제공한다.

### 4.2 공중 화면 생성

하단의 `screen` 버튼을 누르면 바로가기 아이콘 팔레트가 나타난다. 사용자는 Google, Coupang, Wikipedia, Naver Dict, JSFiddle 중 하나를 선택해 해당 웹 화면을 공중에 생성할 수 있다.

생성된 화면은 사용자의 시선 앞에 배치되며, 여러 개의 화면을 동시에 만들어 다중 작업 환경을 구성할 수 있다.

### 4.3 직접 URL 입력

화면을 선택하면 오른쪽 설정 패널이 나타난다. 사용자는 URL 입력창에 원하는 웹 주소를 입력하고 `Go` 버튼을 눌러 화면 내용을 변경할 수 있다. `https://`를 생략해도 자동으로 보정되어 접근하기 쉽게 설계되어 있다.

### 4.4 화면 조절

선택한 화면은 설정 패널에서 조절할 수 있다.

- `width`: 화면 너비 조절
- `height`: 화면 높이 조절
- `curve`: 화면 곡률 조절

일반 웹 화면은 평면 또는 약한 곡면으로 사용할 수 있고, 동영상 모드에서는 큰 곡면 스크린처럼 확장되어 극장형 시청 경험을 연출한다.

### 4.5 화면 이동 및 삭제

생성된 화면에는 이동 핸들과 삭제 버튼이 제공된다. 사용자는 화면을 원하는 위치로 옮기거나 필요 없는 화면을 제거할 수 있다. 이를 통해 사용자가 직접 자신만의 공간 작업 환경을 구성하는 경험을 제공한다.

### 4.6 동영상 모드

`video mode` 버튼을 누르면 대형 곡면 스크린이 생성되고 예시 동영상이 재생된다. 사용자는 직접 `.mp4`, `.webm`, `.ogg` 형식의 동영상 URL을 입력해 극장형 미디어 감상 화면을 구성할 수 있다.

## 5. 사용 흐름

1. 랜딩 페이지에서 Apple Vision Pro와 VisionSpace Preview의 개념을 확인한다.
2. `체험하기` 버튼을 통해 3D 체험 공간으로 이동한다.
3. 안내 오버레이를 클릭해 공간 체험을 시작한다.
4. 마우스와 WASD 키로 공간을 둘러본다.
5. `screen` 버튼을 눌러 웹 화면을 생성한다.
6. 설정 패널에서 URL, 너비, 높이, 곡률을 조절한다.
7. `video mode`로 대형 곡면 스크린을 생성해 몰입형 미디어 환경을 확인한다.
8. 필요에 따라 Apple 구매 페이지로 이동한다.

## 6. 기술적 특징

- HTML, CSS, JavaScript 기반의 정적 웹 프로젝트
- A-Frame을 활용한 브라우저 기반 3D 공간 구현
- Three.js 텍스처를 활용한 바닥 그라데이션과 Apple 로고 마크 표현
- iframe을 활용한 웹 화면 표시
- video texture를 활용한 3D 공간 내 동영상 스크린 구현
- 반응형 레이아웃을 적용한 소개 페이지
- Apple 제품 페이지의 시각 언어를 참고한 미니멀한 UI 구성

## 7. 프로젝트의 의의

VisionSpace Preview는 공간 컴퓨팅이라는 추상적인 개념을 사용자가 직접 조작 가능한 형태로 바꾼 프로젝트이다. 제품 설명을 읽는 데서 끝나지 않고, 화면을 만들고 배치하고 조절하는 과정을 통해 Vision Pro의 핵심 사용 장면을 체험하게 한다.

특히 구매 전 체험이라는 관점에서, 사용자는 제품이 제공하는 몰입감과 활용 가능성을 미리 상상할 수 있다. 이는 제품 홍보, 전시, 발표, 사용자 교육 자료로 확장하기 좋은 형태의 프로토타입이다.

### 8.3 문제의식

포스터에 짧게 넣을 내용:

- Apple Vision Pro는 새로운 형태의 공간 컴퓨팅 기기이지만, 실제 착용 전에는 사용 경험을 상상하기 어렵다.
- 제품 소개 이미지와 영상만으로는 공중 화면 배치, 몰입형 영상 감상, 다중 작업 흐름을 충분히 이해하기 어렵다.
- 따라서 구매 전 사용자가 직접 조작해볼 수 있는 가상 체험 공간이 필요하다.

### 8.4 해결 방법

- 브라우저에서 실행되는 3D 체험 공간을 제작했다.
- 사용자가 직접 공중 화면을 생성하고 위치와 크기를 조절할 수 있게 했다.
- 웹 화면과 동영상 화면을 모두 지원해 생산성, 검색, 쇼핑, 미디어 감상 흐름을 시연했다.
- 소개 페이지와 체험 공간을 연결해 정보 탐색에서 체험으로 이어지는 구조를 만들었다.
